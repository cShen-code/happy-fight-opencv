# HappyFightOpencv

#### 介绍
基于python，opencv的模板匹配制作的微信小程序

# 安装 opencv 使用清华镜像下载
```powershell
# 安装 opencv 使用清华镜像下载
pip install cv2 -i https://pypi.tuna.tsinghua.edu.cn/simple
pip install cv2 -i https://pypi.tuna.tsinghua.edu.cn/simple
pip install win32gui -i https://pypi.tuna.tsinghua.edu.cn/simple
```
